## Deployment Instructions (Important!)

This project uses Vite. When deploying to services like **Cloudflare Pages**, **Netlify**, or **Vercel**, you MUST configure the build settings correctly:

- **Build Command:** `pnpm run build` (or `npm run build`)
- **Output Directory:** `dist`

**If you see errors like "disallowed MIME type" or "bare specifier", it means you are serving the source code instead of the built application. Please check your output directory setting.**

## Local Development

```bash
pnpm i
pnpm run dev
```
